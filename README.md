## Overview

This repository contains code and materials for studying the recoverability of population-level fair insurance premiums under selection bias, with applications to actuarial fairness, causal inference, and discrimination-aware pricing.[1]

## Contents

- Simulation code.
- Theoretical derivations and numerical experiments.
- Figures and outputs used for the paper.
- Implementations related to aware, unaware, and corrective premiums.
